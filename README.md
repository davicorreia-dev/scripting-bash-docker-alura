# 🚀 Scripting com Bash e Docker | Alura

Repositório criado para armazenar minhas práticas, exercícios e projetos desenvolvidos durante o curso:

📚 **Scripting: automatizando tarefas com Bash e Docker** — Alura

---

## 🧠 Sobre o que estudei

Durante o curso, trabalhei com automação de tarefas utilizando Bash e Docker, passando por conceitos essenciais como:

* 🔹 Criação de scripts Bash
* 🔹 Permissões de execução (`chmod`)
* 🔹 Automação com `crontab`
* 🔹 Estruturas condicionais (`if`, `case`)
* 🔹 Laços de repetição (`for`, `while`)
* 🔹 Criação de funções em Bash
* 🔹 Manipulação de diretórios e arquivos

---

## 🐳 Docker

Também explorei conceitos importantes de containerização:

* 📦 Criação de containers com Docker
* 🛠️ Construção de imagens com Dockerfile
* 🔄 Versionamento de imagens no Docker Hub
* ⚙️ Orquestração de múltiplos containers com Docker Compose
